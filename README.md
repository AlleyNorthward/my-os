# 简介
这部分呢, 会详细地深入讨论计算机`os`底层启动细节. 面前为止, 主要有以下几个层面, 第一是`beginner`, 简单模式, 几百行代码快速了解, 第二是`expert`, 高级模式, 几千行代码写一个操作系统, 第三就是`machinestate`, 用`python`写的计算机启动模拟器, 仿真计算机启动时刻状态, 可视化了解其运行情况. 最后一个就是原理模块了, 讲述底层实现的原理, 深入理解操作系统实现的机理. 后续可能还会添加东西, 但是不会再进行补充了.
